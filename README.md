Student Management System (Python + MySQL)

About

This is a simple Student Management System made using Python and MySQL. I created this project to practice database connectivity and CRUD operations using Python.

The application runs in the command line and allows users to manage student records easily.

---

Features

- Add Student
- View All Students
- Search Student by ID
- Update Student Name
- Update Student Age
- Update Student City
- Delete Student (with confirmation)
- Input Validation
- Error Handling using Exception Handling

---

Technologies Used

- Python 3
- MySQL
- mysql-connector-python

---

Database Setup

1. Install MySQL.
2. Open MySQL Workbench.
3. Run the "sql-python.sql" file.
4. Update the database username and password in "database.py".
5. Run "main.py".
